Install Git

The install process for git is different based on the type of computer you are using. Install information is provided here for Windows & Mac OS. If you are using Linux, it is likely that you already have git and if not can use the package manager to easily install it.

Windows Users

If you are a Windows user, typically you will use PowerShell as your command shell. However, Windows PowerShell does not include git by default. You will need to download and install git from the web. You can get it from the Git-SCM Downloads.

I recommend that you keep the default install settings suggested by the installer, specifically:


This will install a few different options that allow you to work with Git on a Windows computer. You may choose any of them you like, but I would recommend GitBash as it will be closest to what you may see from my examples.

Mac Users

If you are using a Mac, open up a Terminal window and type git. You may find that it is already installed, or if not that it will prompt you to install the XCode Command Line Developer Tools. If it does not, you likely have an older version of the Mac OS and should download and install the tools from the Apple Developer Site. You can also install the full XCode suite from the Apple App Store, but this will take additional space.

GitHub Desktop is not required, we only need the underlying git command, and actually I encourage you not to use the application itself, but there is also now a GitHub Desktop for Mac
