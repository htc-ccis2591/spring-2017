Learning Outcomes

This page collects together all of the “outcomes” associated with individual modules. Outcomes identify what students will know and be able to do if they master the material.

JavaScript Functions

Write basic functions in JavaScript that:

  • take input parameters
  • use return values
  • are stateless

Referencing modules: Functions & Scope

JavaScript Syntax

Write basic JavaScript code using:

  • variables
  • conditional statements (if and if/else)
  • iteration (for and while loops)
  • mathematical operators
  • string concatenation
  • boolean expressions

Referencing modules: Hello JavaScript

Git & Basic Shell Commands

Use the shell to use shell commands to:

  • identify the current working directory
  • view files in the current working directory
  • change the current working directory

Referencing modules: Getting Started

Assessed by:

JavaScript Scope

Explain the differences between ES5 and ES6 as it relates to variable declaration and scope.

Referencing modules: Functions & Scope

JavaScript Tools

Identify the purpose of the following common JavaScript tools:

  • Node
  • Gulp
  • Bable
  • ESLint

Referencing modules: Getting Started

Git & GitHub Workflow

Use git commands and the GitHub workflow to complete an assignment for this course.

  • fork a GitHub project
  • obtain a local copy of a repository and its files from GitHub
  • stage modified files to be versioned and updated
  • version the changes to the repository files
  • update GitHub with the revised version of the repository files
  • view the published website on GitHub
  • create a pull-request for the work to be graded

Referencing modules: Getting Started
